# vue-todolist-api
Project using Vue3 and DATA API
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### API
DATA: https://jsonplaceholder.typicode.com
